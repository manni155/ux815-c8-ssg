# Challenge 8: Static Site Generator 

Crate a basic two-page website using a static site generator. It's up to you which one you choose (start with Astro or 11ty if you want to stay in the JavaScript family; but there are plenty of others)--use this as an opportunity to explore a new tool. The two-page site can be any topic of your choosing, and include at least one image. Commit and publish your site to Netlify and upload to the public Netlify and GitHub URLs to the C8: Dropbox
